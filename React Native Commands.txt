|***** React Native Important Commands *****|

1: For creating react native project in your system:
npx react-native init ProjectName

2: To check how many devices are connected with your system:
adb devices

3: For cache clean:
npm cache clean --force

4: For start react native app:
npx react-native run-android

npx react-native run-android --port 8088 --deviceId 067682515B103837

5: For installation of firebase:
npm install @react-native-firebase/app
npm install @react-native-firebase/auth
npm install @react-native-firebase/firestore
npm install @react-native-firebase/storage

6: For installation of Redux:
npm i redux
npm i react-redux
npm i redux-thunk

7: To open console for React Native:
Ctrl + m and click debug.

8: For installation of React Native Developer Tools

1: npm i react-devtools
2: And then add this in the package.json file inside the end of scripts object.
"devtools": "react-devtools"
3: npm run devtools

9: For installation of React Native Radio Buttons package.
npm i react-native-simple-radio-button

10: For installation of React Native Async Storage.
npm i @react-native-async-storage/async-storage

11: For installation of redux persist library
npm i redux-persist
